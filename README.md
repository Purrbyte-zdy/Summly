# Summly 0.0.1-beta1 - 智能文档摘要工具

> **AI驱动的文档自动化处理解决方案**  
> 批量生成文件摘要并智能重命名，提升知识管理效率

---

## 核心功能

- **批量AI摘要**：支持TXT/DOCX/DOC格式的批量处理
- **语义化重命名**：基于内容生成描述性文件名

## 当前版本状态

**Beta测试阶段** - 0.0.1-beta1  
早期版本可能存在稳定性问题，欢迎反馈改进

### 已知限制

- 部分复杂文档格式解析可能不完整
- 重命名逻辑在特定场景下可能不够精确
- 超大文件处理性能待优化
